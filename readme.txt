== Description ==

Module de livraison depuis la France vers la France et l'international pour le plugin WP e-Commerce.
Module enlèvement sur place.

Au moment du réglement le client pourra choisir entre :
* Lettre Max
* Colis standard (Colissimo, enveloppe à document, pochette à bulle, boite spéciale)
* Chronopost (Classique, prêt-à-expédier)
* Enlèvement sur place (lieu de stockage des marchandises ou boutique physique par exemple)

Vous pouvez choisir dans l'interface si vous emballez vos colis par vos soins (tarif standard) ou si vous utilisez des produits prépayés de la poste (enveloppe à bulle,...)

Pour un développement personalisé et une aide à l'installation merci de remplir le formulaire qui se trouve sur la page : http://wpcb.fr/plugin-livraison-france/

== Installation ==

1. Envoyer `wp-e-commerce-livraison-france` vers le dossier `/wp-content/plugins/`
2. Activer le plugin sur le menu 'Extensions' de WordPress
3. Aller sur 'Réglages -> Boutique -> Livraison' et editer -> 'Livraison France'
4. Documentation supplémentaire sur : http://wpcb.fr/plugin-livraison-france (en Français)

== Frequently Asked Questions ==
